# **产品选择原则**

## 产品选择专业原则

### 2.0 通用规则（所有资产类别先看这4条）
- **先确定“买的是什么”**：指数基金要先确认跟踪的指数；债券基金要先确认久期与信用暴露；黄金要先确认是否为“价格工具”还是“主题股票”。
- **费用口径统一**：
  - **年费合计** = 管理费 + 托管费 + 销售服务费（如有；C类常见）
  - **交易费用** = 申购费 + 赎回费（按持有期限）+（若场内）佣金/价差
- **输出结构固定**：每个资产类别只保留 **主选1只 + 备选1-2只**。
- **信息源优先级**：基金合同/招募说明书 ＞ 定期报告（季报/年报）＞ 第三方页面。

4. **定主选/备选**：主选1只、备选1-2只；以后按“产品更换触发条件”处理，不随意换。

## 四、专业理财师三定律

### 定律一：不追求最优解原则
- **核心理念**：费用最低≠最好，需平衡跟踪精度与规模稳定性
- **执行标准**：选择费率前20%且跟踪误差前30%的产品
- **避免陷阱**：过度优化导致频繁换基，增加摩擦成本

### 定律二：第二选择验证原则
- **核心理念**：永远准备2个备选产品，首选变质时立即切换
- **执行标准**：每个资产类别维护1个主选+2个备选清单
- **更新频率**：每季度更新一次备选清单

### 定律三：沉没成本无视原则
- **核心理念**：发现更好产品时，立即切换无视已有持仓成本
- **执行标准**：预期收益提升>0.2%/年即可考虑切换
- **决策模型**：切换成本 vs 未来收益差异的NPV分析


# **周期化执行清单**

## 注意

- 检查周期与KPI监控：
    - **月度**：小复盘（投入/收益率、目标权重偏离、是否触发定投或纠偏）；**定投执行率≥95%**。
    - **季度**：产品体检（跟踪误差、费率/规模变化、经理/风格漂移）、对照基准与同类；**估值口径复盘**。
    - **每4个月**：再平衡（偏离≥8%或到期校准），检视风险预算与现金缓冲；**优先用净流入纠偏**。
    - **年度**：目标与资产配置复核（收入/支出/学业阶段变化，必要时调整权益比例与定投额度）；**定投规模与收入同步上调**。
    - **触发式**：出现重大回撤达阈值（如-25%）→**仅被动再平衡，禁止加杠杆**；单资产超配≥目标+5%→逢高减回；估值分位<20%→在上限内优先增配。

- 避免集中在单一指数或单一基金公司。
- 坚持至少3-5年持有周期，不频繁换基。
- 若遇突发支出优先动用机动金，不动用长期资产。
- 停止条件：学业/收入不稳定或出现大额支出预期时，暂停提升权益仓位，优先补充机动金。

- 卫星仓（对应方案分析-卫星仓建议）：
  - **总占比≤10%、单主题≤5%**，事先写明止盈/止损与期限；执行上严格小仓位试错。
  - 可选配置：行业/因子（如红利/价值/成长），仅在充分理解回撤与波动后尝试。
  - 风险控制：避免与核心持仓高度相关，定期检查风格漂移。

### 4）触发式（出现条件就执行）
- [ ] **回撤触发**：若触及风险预算（如-25%）→ 仅做被动再平衡，禁止加杠杆。

### 5）入场与退场策略

> 说明：这里的“退场”不是清仓离场，而是**纪律化减仓/暂停权益新增/回到目标权重**。

**入场（建仓）**
真正需要的不是“预测明天涨跌”，而是一套不依赖实时判断也能长期有效的入场规则：
- 对宽基（沪深300/中证500/中证1000）
如果一定要“择时”：只用已有的“估值分层”做轻度调整——
估值分位 <30%：当月按1.5x（多买）
30%–70%：1.0x（正常买）
>70%：0.5x（少买）

- 对债券基金/固收
一般不需要择时：你用的是短久期、控信用的“稳健底仓”，更关键是久期/信用而不是入场点位。按定投即可。

- 对现金池（货币基金）
完全不择时：这是给应急金与“再平衡弹药”的，追求的是可用性和稳定性。

- 风控退场规则：当组合回撤触及阈值时，停止临时加码，新增资金优先进入现金池+债券，直到回撤收敛后再恢复正常节奏——这比“赌一个完美入场点”更可控。

- **初始资金建仓**：分3批（T/T+7/T+14）；触发“周跌≥2%提前/周涨≥5%顺延”则按规则执行。
- **后续入场（每月定投）**：不做主观择时；按D2执行既定定投与估值分层（低估1.5x/中性1.0x/高估0.5x）。

**退场（减仓/暂停）**
- **再平衡式退场**：权益（宽基）权重偏离目标≥8%时，优先用净流入纠偏；仍超标则在再平衡窗口内卖出回到目标权重。
- **风险预算退场**：当组合回撤触及-25%：
  - **停止任何临时加码/额外加仓**；
  - **新增资金优先进入现金池与债券基金**，直至回撤收敛到>-20%后再恢复正常权重（机械规则，避免情绪化）。
- **生活事件退场**：若出现收入不稳定/明确大额支出预期 → 暂停提升权益仓位，新增资金优先补充现金池。

### 6）产品更换策略

**A. 主选/备选机制**
- 每个资产类别固定维护：**主选1只 + 备选1-2只**（备选必须提前按量化表验过）。

**B. 触发阈值与动作**
- **黄灯（观察）**：进入观察期，不立刻切换。
  - 观察期：**1个月**（避免单次异常导致误判）
  - 核验来源：基金合同/招募说明书 + 最近2期季报/年报（至少2个独立来源）
- **红灯（强制）**：立即停止旧产品新增投入，启动替换流程。

**C. 替换流程（含反复横跳防护）**
1. **停止旧产品新增**：红灯触发当月起旧产品不再加仓；黄灯期间可“暂停新增1个月”等待结论（减少噪声）。
2. **新旧并行**：将旧产品的新增资金转投到备选产品，**并行3个月**验证稳定性。
3. **切换完成**：在赎回费更友好的时间点（或下一个再平衡窗口）逐步退出旧产品，避免一次性卖出造成不必要摩擦成本。
4. **冷静期规则**：同一资产类别**6个月内最多切换1次**（除非红灯强制）。

- 证据来源：报告/公告链接或页码：____
- 决策：观察/并行/切换完成
