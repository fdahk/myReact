# 加减仓策略：定投拆分 + 回撤分档加仓 + 再平衡止盈

> 目标：在不做主观预测的前提下，降低“买在单点”的体感风险，并用机械规则在大幅回撤时提高长期期望收益；上涨阶段通过再平衡锁定一部分收益、控制组合风险。
>
> 适用：宽基指数为核心（沪深300/中证500/中证1000），长期投资；**最大回撤容忍度约 -25%**；不做“跌了就卖、等更低再买”的传统止损。

---

## 0）核心原则（务必先读）

- **只做“买得更聪明”，不做“卖得更聪明”**：下跌不靠卖出择时；上涨只靠再平衡卖“超配部分”。
- **预算守恒优先**：额外加仓必须有明确资金来源与上限，避免“越跌越加 → 现金流失控”。
- **一切动作可复盘**：每次加仓/减仓都要写明触发原因与执行金额（见本文末记录模板）。

---

## 1）名词与口径（统一口径，避免执行时争议）

### 1.1 基础定投 vs 额外加仓

- **基础定投**：无条件执行的固定频率买入（永远不断，除非个人现金流危机）。
- **额外加仓**：在“回撤分档”触发时，额外投入的那一部分（有上限、有冷却期）。

### 1.2 回撤口径（标准口径）

- **回撤（DD）**：从“近52周最高点”到当前的跌幅：
- 触发对象优先顺序（从推荐到可替代）：
  1. **权益组合/权益账户的净值曲线**（最贴近你的真实持仓）
  2. **核心宽基的加权指数**（例如你权益内部的58/30/12加权）
  3. **单一指数（沪深300等）**（仅当你实在无法做加权/组合追踪）

---

## 2）方案执行结构

### Step A：设置“基础定投频率”（解决错过点位）

默认推荐（从易到稳）：
- **推荐默认**：每周一次（4次/月均摊）

### Step B：基础定投（永远不断）

- 每次基础定投金额 = 当月权益预算 / 当月基础定投次数
- 基础定投按你既定的宽基内部结构执行（例如 300/500/1000 = 58/30/12）
- 同时使用“估值分层”，它只改变当月三只宽基之间的分配，不改变基础定投“是否执行”的纪律

### Step C：回撤分档“额外加仓”（核心增强）

#### 2.1 触发档位（标准三档）

当回撤达到下列档位时，允许额外加仓（叠加，但需满足冷却期与上限）：

- **档位1（轻度）**：\(DD \le -10\%\) → 当月权益投入上浮到 **1.2×**
- **档位2（中度）**：\(DD \le -20\%\) → 当月权益投入上浮到 **1.5×**
- **档位3（重度）**：\(DD \le -30\%\) → 当月权益投入上浮到 **2.0×**

> 解释：这里的“上浮”是相对当月权益预算的倍率（例如基础权益预算为E，则档位2目标权益投入=1.5E）。

#### 2.2 资金来源与上限（避免失控）

额外加仓资金 **只允许** 来自：
- **现金池（货币基金/活期）**：默认来源
- **当月未用完预算**：若你有结余可追加，也允许

额外加仓上限：
- **每月最多额外动用现金池余额的 20%**
- 若现金池不足以支撑目标倍率，则**按上限执行，不追求满倍率**

#### 2.3 冷却期（防止震荡反复触发）

- 每触发一次“额外加仓”，进入 **冷却期 2 周**：冷却期内不再触发下一次额外加仓。
- 冷却期结束后，若仍满足更深档位条件，才允许触发下一档。

#### 2.4 退出/恢复（从“加仓模式”回归常态）

- 当回撤从档位区间“解除”（例如从≤-20%回到>-20%），**下一月**恢复到较低档位倍率或恢复基础预算。
- 额外加仓不是永久提高投入：它是一种“逆周期、有限子弹”的规则。

### Step D：上涨只做“再平衡止盈”（不做固定收益止盈）

#### 2.5 再平衡触发

建议周期：**每4个月**（或季度也可以）检查一次；并使用偏离阈值触发：

- 若**权益权重**相对目标权重 **超配 ≥ +8%**：
  - 卖出超配部分（只卖“超出目标的那一截”，保留核心底仓）
  - 资金转入：优先补充固收/现金池（恢复目标权重）
- 若权益权重 **低配 ≤ -8%**：
  - 优先用新增资金纠偏；必要时从固收/现金池转回补足

> 说明：再平衡是“止盈”的标准替代：它不猜顶，只把风险暴露拉回你能承受的水平。

---

## 3）防守模式（替代止损：控制未来风险，而不是低位卖出）

当**组合最大回撤触及 -25%**时，进入防守模式：

- **停止所有“额外加仓”**（回撤分档加仓暂停）
- **基础定投不必全停**，但权益新增可以降档：
  - 建议：当月权益新增降为基础预算的 **50%**（另一半转入现金池/固收）
- 退出防守模式的默认规则（机械化）：
  - 当组合回撤收敛到 **>-20%** 时，恢复常态（基础定投 + 回撤分档规则重新生效）

> 目的：在最差阶段降低“继续恶化”的概率，但不把自己彻底踢出市场（避免踏空反弹）。

---

## 4）结合“估值分层定投”

同时使用：
- **回撤分档**：决定“本月权益总投入”倍率（1.0× / 1.2× / 1.5× / 2.0×）
- **估值分层**：只决定“本月权益内部三只宽基如何分配”（例如低估1.5x/中性1.0x/高估0.5x后再做预算守恒缩放）

## 5）执行清单（每月/每周

### 每周（基础定投）
- [ ] 固定交易日执行基础定投（每周一次）

### 每月（月初一次）
- [ ] 计算当月“基础权益预算E”（来自你主计划的月度分配）
- [ ] 查看“近52周回撤DD”（指数/权益组合）
- [ ] 若触发回撤档位：按规则计算“额外加仓金额”（不超过现金池20%）
- [ ] 记录本月是否触发（见记录模板）

### 每4个月（或季度）
- [ ] 检查权益权重偏离是否 ≥ 8%
- [ ] 超配则卖出超配部分并转入固收/现金池（再平衡止盈）

### 触发式
- [ ] 组合回撤 ≤ -25%：进入防守模式（暂停额外加仓、降低权益新增）
