# 财富扩展

---

## 一、主业收入最大化

这是最稳定、风险最低的财富来源。

### 1. 技术深耕
- **成为领域专家**：选择一个有前景的方向深耕（AI/ML、云原生、安全等）
- **技术栈选择**：关注高薪技术栈（Rust、Go、Kubernetes、大数据等）
- **持续学习**：每年投入时间学习新技术，保持市场竞争力

### 2. 跳槽策略
- 合理跳槽是涨薪最快的方式（通常涨幅 20-50%）
- 保持 2-3 年一次的跳槽节奏
- 关注大厂、外企、独角兽公司的机会

### 3. 晋升路线
- **技术路线**：工程师 → 高级工程师 → 架构师 → 技术专家
- **管理路线**：工程师 → Tech Lead → 技术经理 → 技术总监

### 4. 高薪行业
- 金融科技（FinTech）
- 人工智能（AI）
- 量化交易
- 区块链
- 外企 / 远程工作（美元收入）

## 二、副业收入

### 1. 独立开发
- **SaaS 产品**：开发解决特定问题的订阅制软件
- **移动应用**：App Store / Google Play 上架应用
- **浏览器插件**：Chrome 扩展程序
- **开发者工具**：CLI 工具、IDE 插件、开源项目商业化

### 2. 接外包/私活
- 国内：猪八戒、程序员客栈、码市
- 国外：Upwork、Toptal、Fiverr（美元结算，收入更高）
- 注意：需要平衡精力，避免影响主业

### 3. 技术咨询
- 为中小企业提供技术咨询服务
- 架构设计、代码审查、技术选型
- 时薪可达 500-2000+ 元

### 4. 知识变现
- **技术博客**：广告收入、付费专栏
- **在线课程**：极客时间、掘金小册、B站付费课程
- **出版书籍**：技术书籍版税
- **付费社群**：知识星球、小报童

### 5. 自媒体
- 技术视频（B站、YouTube）
- 技术公众号
- 播客节目

## 三、投资理财

### 1. 基础配置
- **应急资金**：保留 3-6 个月生活费的现金
- **保险**：医疗险、重疾险、意外险（程序员是高危职业）

### 2. 稳健投资
- **指数基金定投**：沪深300、中证500、标普500
- **债券基金**：降低组合波动
- **国债逆回购**：闲置资金短期理财

### 3. 进阶投资
- **股票投资**：需要学习财务知识，建议从行业 ETF 开始
- **REITs**：房地产信托基金，门槛低的房产投资
- **港美股**：更多优质科技公司标的

### 4. 投资原则
- 不懂不投
- 不要把所有鸡蛋放一个篮子
- 长期持有，避免频繁交易
- 投资前先学习，推荐书籍：《小狗钱钱》《富爸爸穷爸爸》《指数基金投资指南》
