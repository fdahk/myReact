## 建议阅读顺序

1. 先看 [`review.md`](./review.md)，明确执行顺序和主项目目标。
2. 再看 [`面经分类整理.md`](./面经分类整理.md)，按高频主题回查基础专题。
3. 接着训练 [`项目表达与自我介绍.md`](./项目表达与自我介绍.md)，并选定一个项目规划文档深挖。
4. 最后补高阶专题和 `code` 目录，把项目经验、场景题和代码训练串起来。
