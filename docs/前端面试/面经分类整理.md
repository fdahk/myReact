# 面经分类整理

## 使用方式

这个文档不是新的题库，而是把真实面经按主题归档，方便你快速判断：哪些问题已经有对应训练文档，哪些高阶能力还需要补齐。

你可以把它当成三件事的入口：

- 高频面试题索引
- 训练文档映射表
- 查漏补缺与调整训练顺序的参考

---

## 一、高频主题总览

从当前收集到的题目看，出现频率最高的是下面 8 类：

1. 自我介绍、实习经历、项目介绍、职业规划
2. `React` 基础、Hooks、组件通信、渲染机制
3. `React Fiber`、调度、优先级、`React 19`
4. 浏览器渲染、白屏、重排重绘、性能优化、`Worker`
5. 网络、缓存、登录态、跨域、流式协议
6. 工程化、构建、`CI/CD`、发布治理、灰度发布
7. `AI` 前端链路、`BFF`、`Node.js / Python` 分工、工具调用
8. 算法、手写题、场景题与问题排查

## 三、本轮补齐的高阶专题

这轮面经收口后，建议你把下面 4 个专题视为“项目深度补位”：

- `BFF` 与前端网关：补的是接口编排、流式转发、会话和协议裁剪，不再只会讲页面层。
- 国际化与多地区发布：补的是多语言、多地区、多时区与 `SSR/SEO` 协作，不再把国际化理解成纯翻译。
- 监控埋点与前端可观测性：补的是错误、性能、接口、版本和行为数据的关联分析能力。
- 灰度发布与回滚：补的是上线风险控制、分桶放量、回滚止损和版本治理能力。

## 四、建议训练顺序

如果按真实面试频率和项目收益排序，建议这样练：

1. `React` + 浏览器性能
2. `JavaScript` + 网络 + `TypeScript`
3. 工程化 + 可观测性 + 灰度发布
4. `BFF` + `AI` 应用前端专题
5. 国际化 + `SSR / Next.js`
6. 算法与手写
7. 项目表达与模拟面试
